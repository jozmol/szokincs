# Kínai-Magyar-Angol Szótár Gyakorló

## 📚 Leírás
Interaktív alkalmazás kínai, magyar és angol szavak gyakorlásához.

## 🚀 Funkciók
- 500+ szó 12 kategóriában
- Hangos kiejtés
- Többnyelvű felület

## 🛠️ Használat
1. Nyisd meg a `index.html` fájlt böngészőben
2. Válassz kategóriát
3. Gyakorolj!
